# 🤖 Machine Learning Model

## What model is used?

We use XGBoost for prediction.

---

## What does it predict?

- 1 → Approved
- 0 → Rejected

---

## Important features

- Income
- Expenses
- Loan amount
- Credit score

---

## Why ML is used?

ML helps learn patterns from past data.

Example:
- High expenses → higher risk
- Low credit score → rejection

---

## Problem with ML alone

ML is not enough.

So we add:
👉 Rule-based system for business logic
