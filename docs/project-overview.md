# 🏦 Project Overview

## What is this project?

This is an AI-powered Loan Decision System.

It helps decide:
- Should a loan be approved?
- What is the risk level?
- Why was the decision made?
- What should the user improve?

---

## How it works (simple)

1. User enters financial details
2. Backend processes data
3. ML model predicts approval/rejection
4. Rule engine adds business logic
5. System generates explanation
6. Simulation suggests improvements

---

## Technologies used

- Python (FastAPI)
- Machine Learning (XGBoost)
- React (Frontend UI)
- SHAP (Model explainability)
- Render (Deployment)

---

## Final Output

- Loan decision (Approved / Rejected)
- Risk level
- Explanation
- Simulation (what to improve)
